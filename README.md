# EducaLista

**EducaLista** é um aplicativo desenvolvido em React Native para gerenciar a presença escolar e contatos dos pais. Através de uma interface amigável, você pode registrar a presença dos alunos e gerenciar uma lista de contatos dos pais.

**EducaLista** is an application developed in React Native to manage school attendance and parents' contacts. Through a user-friendly interface, you can record student attendance and manage a contact list of parents.

## Descrição / Description

EducaLista é uma solução intuitiva para escolas e professores. Permite o registro de presença dos alunos e o gerenciamento de contatos dos pais com eficiência e facilidade. Utiliza AsyncStorage para armazenamento local.

EducaLista is an intuitive solution for schools and teachers. It allows for recording student attendance and managing parents' contacts efficiently and easily. It uses AsyncStorage for local storage.

## Funcionalidades / Features

- **Marcar Presença / Mark Attendance:** Registre a presença dos alunos com data e nome.  
  Record student attendance with date and name.
- **Gerenciar Contatos / Manage Contacts:** Adicione, edite e exclua contatos dos pais.  
  Add, edit, and delete parents' contacts.
- **Visualizar Presença / View Attendance:** Filtre a lista de alunos por data para verificar a presença em dias específicos.  
  Filter the student list by date to check attendance on specific days.
- **Interface Intuitiva / Intuitive Interface:** Design clean e fácil de usar.  
  Clean and easy-to-use design.

## Instalação / Installation

Para rodar o projeto localmente, siga estes passos:  
To run the project locally, follow these steps:

1. **Clone o repositório / Clone the repository:**
    ```bash
    git clone https://github.com/seuusuario/educalista.git
    ```
2. **Navegue até o diretório do projeto / Navigate to the project directory:**
    ```bash
    cd educalista
    ```
3. **Instale as dependências / Install the dependencies:**
    ```bash
    npm install
    ```
4. **Inicie o aplicativo / Start the app:**
    ```bash
    npm start
    ```
   Use o Expo Go no seu dispositivo ou um emulador para visualizar o aplicativo.  
   Use Expo Go on your device or an emulator to view the app.

## Uso / Usage

1. **Iniciar o Aplicativo / Start the App:** Após iniciar o aplicativo, você verá a tela inicial com opções para marcar presença, acessar a lista telefônica e visualizar as notas dos alunos.  
   After starting the app, you will see the home screen with options to mark attendance, access the contact list, and view student grades.
2. **Adicionar Presença / Add Attendance:** Navegue para a seção de presença e registre a presença dos alunos.  
   Go to the attendance section and record student attendance.
3. **Gerenciar Contatos / Manage Contacts:** Acesse a lista telefônica para adicionar ou editar contatos dos pais.  
   Access the contact list to add or edit parents' contacts.
4. **Visualizar Presença / View Attendance:** Use o filtro de data para visualizar a presença dos alunos em dias específicos.  
   Use the date filter to view students' attendance on specific days.
